# React Inbox Layout Application

A React-based Inbox Layout Application developed using React, TypeScript, Axios, and MockAPI. The application demonstrates component-based architecture, API integration, state management, reusable UI components, custom hooks, and responsive design principles.

## Features

* Inbox View
* Trash View
* Create Record using Modal Popup
* Search Functionality
* Soft Delete (Move to Trash)
* Permanent Delete from Trash
* Refresh Functionality
* Responsive Grid Layout (Desktop, Tablet, Mobile)
* Reusable Components
* API Integration using Axios
* Custom Hook (useDebounce)

## Tech Stack

* React
* TypeScript
* Axios
* MockAPI
* CSS
