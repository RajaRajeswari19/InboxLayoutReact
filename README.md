# React Mail Application

A React Mail Management Application developed using React, TypeScript, Axios and MockAPI.

## Features

- Inbox View
- Trash View
- Create Email
- Search Email
- Soft Delete
- Permanent Delete
- Responsive Grid Layout
- Custom Hook (useDebounce)

## Tech Stack

- React
- TypeScript
- Axios
- MockAPI
